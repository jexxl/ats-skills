# ats-algorithm-cover

生成自主式交通系统（ATS）算法模型封面卡片，用于云计算平台展示。

## 使用方式

安装 skill 后，在对话中说“帮我生成算法封面”“生成算法模型封面卡片”或“批量生成 cover.png”即可触发。Agent 会引导你完成以下步骤：收集或推断封面字段 → 生成 `cover.html` → 渲染透明高分辨率 `cover.png` → 检查尺寸、DPI、透明背景和 Mermaid 流程图。

## 依赖工具

- `uv`：用于运行封面渲染脚本

## 必需字段

- 算法类型：基础算法或专用算法；也可由算法编号中的 `-J-` 或 `-Z-` 推断
- 功能描述：一句不超过两行的模型主要功能描述，并指定或允许选择一个加粗关键词
- 流程节点：2-6 个流程节点，或一段可转换为 Mermaid `flowchart LR` 的流程描述
- 模型服务场景：用于页脚的服务场景文本
- 主要编程语言：例如 `Python`、`C++`
- 输出位置：目标 `cover.html` 和 `cover.png` 路径，或一个可写的算法目录

## 交付物结构

```text
{算法目录}/
├── cover.html
└── cover.png
```

## 人工审查

- 算法类型、功能描述、流程节点、服务场景和主要编程语言是否准确？
- `cover.png` 是否为透明背景，圆角外是否保留 alpha？
- PNG 分辨率和 DPI 是否满足要求，默认 DPI 是否为 300？
- Mermaid 流程图是否渲染为节点和箭头，而不是源码文本？
- 功能句、服务场景和语言标签是否没有溢出或遮挡？
